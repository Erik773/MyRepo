Предназначен для просмотра списка всех зарегистрированных серверов облачной подписи.

В списке выводится:

  • Наименование - полное наименование сервера облачной подписи; 
  • Адрес сервера – содержит web-адрес подключения к серверу облачной подписи; 
  • Комментарий – дополнительное описание о сервере.

Просмотр данных о сервере

  • С помощью команды Изменить контекстного меню по правой кнопке мыши или меню Еще можно открыть данные о сервера облачной подписи.
